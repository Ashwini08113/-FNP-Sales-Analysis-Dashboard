
# 📊 FNP Sales Analysis Dashboard

### Data Analysis Project | Excel | Business Intelligence | KPI Reporting

An interactive **Sales Performance Dashboard** built using **Microsoft Excel** to analyze 1000+ transactional records and generate actionable business insights.

This project demonstrates my ability to clean, transform, analyze, and visualize data to support strategic business decisions.

---

# 📌 Problem Statement

The business lacks a centralized reporting system to:

* Monitor overall sales performance
* Identify high-performing products and occasions
* Analyze seasonal revenue trends
* Track delivery performance
* Understand customer spending behavior
* Identify top-performing cities

Without structured analysis, decision-making becomes reactive rather than data-driven.

### 🎯 Objective:

To build a dynamic and interactive dashboard that provides real-time insights into sales, customer, product, and operational performance.

---

# 🔄 Project Workflow

### 1️⃣ Data Collection

* Imported raw transactional sales dataset (1000+ records).

### 2️⃣ Data Cleaning

* Removed duplicates
* Handled missing values
* Standardized date formats
* Verified revenue and delivery time fields

### 3️⃣ Data Transformation

* Created calculated columns:

  * Delivery Time
  * Month Name
  * Order Hour
* Structured dataset for Pivot Table analysis

### 4️⃣ Data Modeling & KPI Creation

* Defined business KPIs:

  * Total Revenue
  * Total Orders
  * Average Customer Spend
  * Average Delivery Time

### 5️⃣ Data Analysis

* Aggregated revenue by:

  * Occasion
  * Category
  * Month
  * Hour
  * City
* Identified top-performing products

### 6️⃣ Dashboard Development

* Designed interactive layout
* Created Pivot Charts
* Added slicers for dynamic filtering
* Built KPI summary cards

### 7️⃣ Insight Generation

* Interpreted trends
* Identified seasonal spikes
* Highlighted operational performance gaps

# 📈 Dashboard Insights & Analysis

### 1️⃣ Revenue by Occasion

* Anniversary and Raksha Bandhan drive strong revenue.
* Clear festive sales dependency identified.

### 2️⃣ Revenue by Category

* Colors and Soft Toys show high performance.
* Indicates strong gifting preferences.

### 3️⃣ Revenue by Hour (Order Time)

* Peak order times observed during afternoon/evening.
* Useful for staffing and promotional timing.

### 4️⃣ Revenue by Month

* February and August show revenue spikes.
* Strong seasonal demand pattern detected.

### 5️⃣ Top 5 Products by Revenue

* Identifies highest revenue-generating SKUs.
* Supports focused promotional strategy.

### 6️⃣ Top 10 Cities by Orders

* Certain cities contribute significantly to total order volume.
* Enables regional growth strategy planning.
